# 🚀 دليل النشر — JOKER ELITE SPX Bot

```
joker_deploy/

├── requirements.txt    ← المكتبات
├── Procfile            ← أمر التشغيل
├── render.yaml         ← إعداد Render
└── templates/
    └── index.html      ← الموقع
```

---

## الطريقة الأسهل: Render.com (مجاني)

### الخطوة 1 — رفع الملفات على GitHub
1. اذهب إلى https://github.com وأنشئ حساباً مجانياً
2. أنشئ repository جديد باسم `joker-elite-bot`
3. ارفع جميع الملفات (drag & drop في الواجهة)

### الخطوة 2 — نشر على Render
1. اذهب إلى https://render.com وسجّل دخول بحساب GitHub
2. اضغط **"New Web Service"**
3. اختر الـ repository الذي أنشأته
4. Render سيكتشف الإعدادات تلقائياً

### الخطوة 3 — متغيرات البيئة (مهم!)
في لوحة Render، اذهب إلى **Environment** وأضف:
```
BOT_TOKEN  =  توكن_البوت_الجديد_من_BotFather
CHAT_ID    =  معرف_القناة_أو_المجموعة
```

### الخطوة 4 — التشغيل
اضغط **Deploy** وانتظر 2-3 دقائق
ستحصل على رابط مثل:
```
https://joker-elite-bot.onrender.com
```

## ملاحظات مهمة

### الخطة المجانية على Render:
- ✅ تعمل بشكل كامل
- ⚠️ تنام بعد 15 دقيقة من عدم الاستخدام
- ⚠️ للـ Auto Scan المستمر، استخدم خطة مدفوعة ($7/شهر)
  أو ضع رابط الموقع على UptimeRobot.com (مجاني) لإبقائه يقظاً

### UptimeRobot للإبقاء على الموقع يقظاً:
1. سجّل في https://uptimerobot.com
2. أضف Monitor من نوع HTTP(S)
3. ضع رابط موقعك
4. اضبط الفاصل الزمني على 5 دقائق

---

## استخدام الموقع بعد النشر

1. افتح الرابط في متصفحك
2. أدخل **Bot Token** الجديد (من BotFather)
3. أدخل **Chat ID** قناتك
4. اضغط **"اختبار التوكن"** للتأكد
5. اضغط **"تحليل الآن"** للحصول على إشارة فورية
6. اضغط **"✈️ إرسال لتيليقرام"** لإرسالها
7. شغّل **Auto Scan** للإرسال التلقائي كل 5 دقائق
