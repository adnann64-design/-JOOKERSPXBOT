## 🎯 خطوات النشر على Render.com (5 دقائق)

### الخطوة 1: إنشاء حساب Render
1. اذهب إلى: https://render.com
2. اضغط **"Get Started"**
3. سجل باستخدام حساب GitHub الخاص بك

### الخطوة 2: ربط Repository
1. في لوحة Render، اضغط **"New +"**
2. اختر **"Web Service"**
3. اضغط **"Connect GitHub"** (إذا لم تكن متصلاً)
4. ابحث عن: `-JOOKERSPXBOT`
5. اضغط **"Connect"**

### الخطوة 3: إعدادات الخدمة
Render سيكتشف الإعدادات تلقائياً من `render.yaml`، لكن تأكد من:

```
Name: joker-elite-spx-bot
Region: اختر أقرب منطقة لك
Branch: main
Runtime: Python 3
Build Command: pip install -r requirements.txt
Start Command: gunicorn app:app --workers 2 --timeout 120
```

### الخطوة 4: متغيرات البيئة (مهم جداً!)
في قسم **"Environment Variables"**، أضف:

#### 🔑 BOT_TOKEN
1. افتح Telegram
2. ابحث عن [@BotFather](https://t.me/BotFather)
3. أرسل `/newbot`
4. اتبع التعليمات لإنشاء بوت جديد
5. احصل على التوكن مثل: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
6. في Render، أضف:
   - **Key**: `BOT_TOKEN`
   - **Value**: التوكن الذي حصلت عليه

#### 🆔 CHAT_ID
**للقناة:**
1. أضف البوت كـ Admin في قناتك
2. ابحث عن [@userinfobot](https://t.me/userinfobot)
3. أرسل رابط القناة للبوت
4. احصل على Chat ID (سيكون سالب مثل: `-1001234567890`)

**للمجموعة:**
1. أضف البوت للمجموعة كـ Admin
2. استخدم [@userinfobot](https://t.me/userinfobot)
3. أرسل رسالة في المجموعة
4. احصل على Chat ID

6. في Render، أضف:
   - **Key**: `CHAT_ID`
   - **Value**: معرف القناة/المجموعة

### الخطوة 5: النشر
1. اضغر **"Create Web Service"**
2. انتظر 2-3 دقائق حتى يكتمل النشر
3. سترى رسالة **"Live"** باللون الأخضر
4. احصل على رابط موقعك: `https://jookerspxbot.onrender.com`
